# Haftalık Aksiyon Takibi

React tabanlı haftalık aksiyon takip uygulaması.

## Özellikler

- Yeni aksiyon ekleme formu
- Haftalık ve tür bazında gruplanmış aksiyon görüntüleme
- TreeView yapısı ile aksiyonları organize etme
- Metin dosyalarından combobox veri yükleme

## Kurulum

```bash
npm install
```

## Çalıştırma

```bash
npm start
```

Uygulama `http://localhost:3000` adresinde açılacaktır.

## Veri Dosyaları

- `src/data/weeks.txt` - Hafta listesi
- `src/data/types.txt` - Aksiyon türleri

Bu dosyaları düzenleyerek combobox seçeneklerini güncelleyebilirsiniz.
